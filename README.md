# CASA COYOTES

Sitio web de Casa Coyotes, destilados de pulque y productos de maguey.

## Descripción
Proyecto desarrollado en Angular utilizando arquitectura standalone.
La aplicación está dividida en secciones, cada una representada por una vista independiente.

## Estructura
La aplicación cuenta con 6 secciones:
- Inicio
- Destilados
- Maguey
- Servicios
- Nosotros
- Pedidos

Cada sección corresponde a un componente dentro de `src/app/pages/`.

## Tecnologías
- Angular (standalone)
- TypeScript
- SCSS
- Angular Router
